Вы случайно распределяетесь по ролям: Продавец или Покупатель.

Каждому будет известна его ценность товара и количество товара. Для Продавца это та минимальная цена, по которой он готов продать товар, и то количество товара которое он готов продать. Для Покупателя это та максимальная цена, по которой он готов купить товар и то количество товара которое он готов купить.

Игроки подают заявки на одну единицу продукции: Продавцы на продажу, Покупатели на покупку. Если заявки пересекаются (какой-то покупатель готов купить дороже чем Продавец продать), то между ними заключается сделка по цене первой (из этих двух) заявки. После чего торги продолжаются, пока не закончится время.

При заключении сделки прибыль для единицы продукции определяется по формулам:

Выигрыш Продавца = Цена покупки - Ценность товара для него,
Выигрыш Покупателя = Ценность товара для него - Цена продажи.
Прибыль от разных единиц суммируется.
