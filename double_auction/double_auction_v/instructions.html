Вы случайно и анонимно распределяетесь на пары. В каждой паре игроки случайно получают роль: Продавец или Покупатель.

Для каждой пары Природа независимо и равновероятно определяет ценность неделимого товара для каждого игрока из диапазона [0;100], при этом для Продавца ценность [0;60], а для Покупателя - [40;100]

Игрок узнает ценность товара для него, но не знает ценности товара для другого.

Игроки одновременно подают свои заявки из диапазона [0, 100].

Заявка Продавца означает минимальную цену, по которой он готов продать товар. Заявка Покупателя означает максимальную цену, по которой он готов купить товар.

Если заявка Продавца больше заявки Покупателя, то сделка заключена быть не может. В этом случае выигрыши обоих игроков равны 0.

Если заявка Продавца меньше или равна заявке Покупателя, то между ними заключается сделка купли-продажи по цене, равной полусумме заявок.

При заключении сделки и назначении цены:

Выигрыш Продавца = Цена - Ценность товара для него,
Выигрыш Покупателя = Ценность товара для него - Цена.
В конце каждого раунда игроки в каждой паре видят ценности товара друг для друга, свои заявки и выигрыши.
